●実行ファイルの位置

\x64\Debug\iai.exe

●制作で特に頑張った部分・アピールポイント

・マジックナンバーを使わないように意識し、コメントを詳しく書き、
　自分以外の人が見ても理解しやすいプログラムを意識して制作したこと。

・仕組みを簡単にし、プログラムを見やすくした。

・プレイした記録を、ソートし、ランキング機能を実装した事。

・ランキング機能を実装し、単純だけど、何回も遊びたくなるように工夫した。

・敵と対戦中は、画面を真っ暗にし、無音にすることで、緊張感を持たせた。

・攻撃をカッコいいと思わせれるように、エフェクトや、効果音を工夫した。

●開発環境 Windows10 / Visual Studio 2017

●制作期間「居合斬り」…約1週間半

●使用したライブラリ DxLib

※ゲームをプレイすると、C直下に「IAI_DATA」というフォルダが作成され、
　プレイした記録が保存されます。
