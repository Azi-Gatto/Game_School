●実行ファイルの位置

\LEGENDSTORY\Debug\LEGENDSTORY.exe

●制作で特に頑張った部分・アピールポイント
・敵を倒したときに、アイテムや装備品をドロップさせる処理を作ったこと。
　"main.cpp"の1995行目でドロップした物の種類を判別させていますが、
　アイテムや武器ごとに設定されているコード番号を変更するだけで、新しい種類の
　ドロップ品や、ドロップするアイテムや装備品の種類を増やしたり減らしたりすることができること。

・装備機能を実装したこと。

・選択肢を管理する「SELECTクラス」を作成したこと。
　特に、選択肢として渡す文字列を可変長にすることで、様々な場面で使用できるようにしたこと。
　DrawScroll処理を使用することで、指定された範囲を超えると、スクロールして描画するようプログラムを
　組んだこと。

・マップを、敵と遭遇するフィールドマップと、遭遇しない街マップの2種類を用意し、
　それぞれ9マップずつ制作したこと。

・マップを管理し、移動する処理を自力で考えたこと。

・ブーストポイントという機能を実装したこと。

・敵のステータス、NPCの描画位置などを簡単に変更できること。

・マップの描画方法を、ゲーム内でマップチップを使用して描画するのではなく、
　一枚の画像として描画することで、複数のマップチップを重ねて使用することが簡単になった。

・装備品、アイテムの一覧をcsvファイルで管理しているため、新しく追加することが簡単にできる。

・ゲーム内で使用するSE、BGMの追加が簡単に行える。

・ライブラリのリファレンスや、C++の基本的な機能以外は、自分で処理を考え、
　一からプログラムを組んだこと。

●開発環境 Windows10 / Visual Studio 2017
